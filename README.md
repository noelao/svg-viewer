# svg-viewer
untuk menampilkan svg ke html dengan express

```bash
npm install express ejs
```

buat folder public/svg/icons lalu tempatkan icons berformat .svg didalamanya.
```
|-- /public
|   |-- /svg
|       |-- /icons
|           |-- icon-1.svg
|           |-- icon-2.svg
|           |-- (dan file svg lainnya...)
|-- /views
|   |-- index.ejs
|-- package.json
|-- app.js
```
